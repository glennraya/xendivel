# Xendivel v3.0.0

> [!IMPORTANT]
> v3.0.0 briefly moved invoice PDF rendering to Typeset.sh. v3.0.1 supersedes that change and restores PDF generation through Spatie Browsershot so Xendivel no longer requires a Typeset.sh subscription.

## Highlights

- Official Laravel 13 support.
- PHP 8.5 support.
- Pest 4 and Orchestra Testbench 10/11 support.

## Upgrade Path

Upgrade to the latest `glennraya/xendivel:^3.0` release and follow the v3.0.1 release notes for Browsershot/Puppeteer setup.

## Verification Notes

- Package metadata should pass `composer validate --strict`.
- Full package suite should pass with `composer test`.
- Focused PDF behavior is covered by `tests/Unit/InvoicePdfTest.php`.
